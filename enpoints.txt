UPORABNIK
POST /user/register -> register user
POST /user/login -> authentication
GET /user/ -> get all users
GET /user/userId -> get specific user
GET /user/friends -> get friends of a user

UPRAVLJANJE SLIK
POST /album/?user_id=...  -> post an album by a specific user
GET  /album/?filter=...(filter is optional, also not sure if will be needed to implement) -> get all albums
GET  /album/{albumId} -> get specific album
